1장 <LinearLayout>
project 만들기
app - res - layout 에서 text로 작업하기
preview 켜놓고 작업하기
화면에 꽉채우도록 색 채워넣기(비율 맞춰가며)

assist & completion : 철자 하나만 쳐도 나머지 명령어가 나오는 행위

사용한 커맨드
ctrl + alt + L : 줄맞추기

사용한 코드
<LinearLayout>
	
	메인에 
	android:orientation="vertical" // 세로로 값은 줄 것이기에

        android:layout_width="match_parent" // 부모에 속한 view 만큼 화면에 꽉 채우기
        android:layout_width="wrap_parent" // 화면안에서 사이즈만큼 크기 조절하기	
        android:layout_width="100dp" // 100dp의 크기로 맞추기
        android:layout_height="0dp" // 크기를 주지 않고 무게를 줘서 비율로 사용할때
        android:layout_weight="3" // 3:3:3:1 같이 비율을 줄때의 3
        android:background="#1091F7"> //색깔 집어넣기

하나의 layout에 또 <LinearLayout>을 추가시켜 배율3으로 쪼갠것 안에 가로로 쪼개기(가로로 나온 이유는 orientation 값이 horizontal 로 default 값으로 들어가있기 때문에)
        android:layout_width="100dp"
        android:layout_weight="1" // 위의 width 값을 0dp로 바꿔주고 무게값을 주어 비율로 꽉채우기 (비율이 1이라면 총 2개의 layout을 추가해서 1:1로 채울 수 있다.)
        android:layout_height="match_parent" // 3으로 쪼갠 칸 안에 세로는 꽉 채우기

2장<RelativeLayout>

사용한 커맨드
ctrl + shift + / : 주석처리

사용한 코드	
<RelateLayout>

        android:id="@+id/layout_1" // 집어넣은 색타일 별 아이디 붙여서 구별하기
        android:layout_width="100dp" 
        android:layout_height="100dp"
        android:layout_alignParentTop="true" // 화면의 위쪽에 배치하기 (alignParent 중에서 고르기)
        android:layout_alignParentLeft="true" // 화면의 왼쪽에 배치하기
        android:layout_toLeftOf="@+id/layout_2" // layout_2 왼쪽까지 끌어서 붙이기 

        toLeftOf(오른쪽의 타일 왼쪽에 붙이기) toRightOf(왼쪽의 타일 오른쪽에 붙이기) above(밑의 타일 위쪽에 붙이기) below(위쪽의 타일 아래쪽에 붙이기)

        android:layout_toLeftOf="@+id/layout_2"
        android:layout_toRightOf="@+id/layout_4"
        android:layout_below="@+id/layout_1"
        android:layout_above="@id/layout_3"
	// 4개를 합치면 센터

        android:background="#5C00FF"></RelativeLayout>

3장 <ContraintLayout> - RelativeLayout보다 발전된 것으로 새로운 빈 템플릿을 추가하면 contraintlayout이 default 값으로 들어간다.

이미지를 drawable 폴더에 넣을 때 이름에 숫자가 들어가거나 - 이 들어가있으면 나중에 오류 발생.
